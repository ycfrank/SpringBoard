    displays user manual, press q to exit

    use long listing reference, all or do not ignore entries starting with .
How do you jump between words in the terminal?
    esc + B, esc + F
How do you get to the end of a line in terminal?
    ctrl + A
How do you move your cursor to the beginning in terminal?
    ctrl + E
How do you delete a word (without pressing backspace multiple times) in terminal?
    ctrl + w
What is the difference between a terminal and shell?
    terminal is program that opens UI to let user interact with shell, shell is the UI to access the OS
What is an absolute path?
    specifies location of file from root directory(complete path)
What is an relative path?
    path related to the current working directory (pwd)
What is a flag? Give three examples of flags you have used.
   flags sets options and passes arguements to the commands the user runs,  -l ,-f , -r
What do the r and f flags do with the rm command?
    remove files/content recursively, ignores non-existant files
